Print fan duct with logo facing up [Tree supports, .4mm nozzle, .2 layer height]
Print shell top on the build plate [Tree supports, .4mm nozzle, .2 layer height]
Use the OEM bottom plate of goggles.

Fan duct is fixed to the the top of the top shell with super glue.

Parts used:
https://www.amazon.com/dp/B09329TYCS?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1
This is a snug fit, be careful of the U.FL connector. It is a tight fit. With the 6" I had to put a loop in the cable at the front of the goggle to take up the slack. 4" may work but I have not tried.
